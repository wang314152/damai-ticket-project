# 大麦网票务系统 - 启动说明

## 当前配置

- **前端预览地址**: http://localhost:4002
- **后端地址**: http://localhost:8081

## 快速启动步骤

### 1. 启动后端
```bash
cd d:/2026/集成/damai-ticket-project/damai-ticket
java -jar target/damai-ticket-1.0.0.jar
```
等待出现 `Started DamaiTicketApplication` 后继续（约30秒）

### 2. 构建前端
```bash
cd d:/2026/集成/damai-ticket-project/damai-ticket-frontend
npm run build
```

### 3. 启动前端预览
```bash
npx vite preview --port 4000
```

### 4. 访问系统
打开浏览器访问: http://localhost:4002

## 演示模式说明

如果前端显示"后端未启动，已切换为演示模式"，请确保：
1. 后端服务已完全启动（检查8081端口）
2. 浏览器访问的是带有代理的地址（如 http://localhost:4002 或 http://localhost:5173）
