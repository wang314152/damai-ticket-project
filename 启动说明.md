# 大麦网票务系统 - 启动说明

## 项目概述

这是一个前后端分离的票务系统：
- **前端**: Vue 3 + Vite (端口: 5173)
- **后端**: Spring Boot 2.7.18 (端口: 8081)
- **数据库**: MySQL

---

## 环境准备

### 1. 安装 Node.js (前端)
- 下载地址: https://nodejs.org/
- 推荐版本: 18.x 或更高
- 安装后验证: `node -v`

### 2. 安装 JDK 17 (后端)
- 下载地址: https://adoptium.net/
- 推荐版本: JDK 17
- 安装后验证: `java -version`

### 3. 安装 MySQL 8.0
- 下载地址: https://dev.mysql.com/downloads/mysql/
- 或使用 phpStudy/宝塔面板等集成环境
- 确保 MySQL 服务已启动

---

## 启动步骤

### 第一步：初始化数据库

1. 打开 MySQL 客户端（如 Navicat、MySQL Workbench 或命令行）
2. 执行初始化脚本：
   ```
   damai-ticket/sql/init.sql
   ```
3. 或者在命令行执行：
   ```bash
   mysql -u root -p < damai-ticket/sql/init.sql
   ```

**默认账号密码**：
- 管理员: `admin` / `admin123`
- 测试用户: `test` / `123456`

### 第二步：启动后端

```bash
cd damai-ticket

# Windows 使用 mvnw.cmd
./mvnw.cmd spring-boot:run

# 或先打包再运行
./mvnw.cmd package
java -jar target/damai-ticket-1.0.0.jar
```

后端启动成功后会显示：
```
Started DamaiTicketApplication in X seconds
```

### 第三步：启动前端

```bash
cd damai-ticket-frontend

# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev
```

前端启动成功后会显示：
```
Local: http://localhost:5173/
```

## 常见问题

### 1. 数据库连接失败
检查 `application.yml` 中的数据库配置：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/damai_ticket
    username: root
    password: 123456
```
确保数据库存在且账号密码正确。

### 2. 端口被占用
- 前端默认端口: 5173
- 后端默认端口: 8081
- 可在对应配置文件中修改

### 3. Maven 依赖下载慢
可在 `pom.xml` 所在目录创建 `settings.xml` 配置阿里云镜像。

### 4. npm 安装依赖失败
```bash
# 清理缓存
npm cache clean --force

# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com
npm install
```
